# 🐍 贪吃蛇游戏 — 经典街机挑战

欢迎来到 **Snake Game**，这是一款经典的贪吃蛇游戏，采用了现代 **Glassmorphism（玻璃拟态）** 界面设计，并加入了许多有趣的功能。

![Snake Game Demo](assets/snake_head.png) *(注意：请将图片上传到 assets 文件夹以正常显示)*

## ✨ 核心功能

*   **🎮 流畅体验：** 使用方向键或 W, A, S, D 键控制蛇的移动。内置移动端专用控制面板。
*   **💎 玻璃拟态 UI：** 采用磨砂玻璃效果和绚丽色彩，营造高端视觉享受。
*   **🔐 登录/注册系统：** 基于浏览器本地存储 (localStorage) 的用户管理系统。
*   **📊 游戏历史：** 记录每位用户的游戏场次、游戏时长以及最高分。
*   **💥 粒子特效：** 贪吃蛇每次进食时都会触发精美的爆炸特效。
*   **📱 全平台适配：** 响应式设计，在电脑、平板和手机上均能完美运行。

## 🚀 核心技术

*   **HTML5 & CSS3：** 现代化的网页结构与样式（原生 CSS）。
*   **JavaScript (ES6+)：** 核心游戏逻辑与模块化管理。
*   **Canvas API：** 高性能 2D 游戏绘图。
*   **Google Fonts：** 使用优雅的 'Outfit' 字体。

## 🛠 如何开始

无需复杂的安装步骤：

1.  直接在浏览器中打开 `index.html` 文件。

## 📂 目录结构

*   `index.html`: 主页面结构。
*   `style.css`: 所有的视觉样式与动画。
*   `app.js`: 主控制器与界面逻辑。
*   `game.js`: 贪吃蛇核心逻辑与规则。
*   `auth.js`: 用户身份验证系统。
*   `history.js`: 游戏记录管理。
*   `assets/`: 图像资源（蛇头、苹果）。
